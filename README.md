# teslasentryjs

Installation steps :

1. Install ffmpeg

   ```bash
   - apt-get install libav-tools
   - apt-get install ffmpeg
   ```

2. Install npm :
3. Install git :

   ```bash
   apt-get install git
   ```

4. Clone repo :

   ```bash
   git clone https://github.com/Kirikou974/teslasentryjs.git
   ```

5. Install web site :

   ```bash
   cd teslasentryjs npm install
   ```

6. Run web site :

   ```bash
   node server.js
   ```
